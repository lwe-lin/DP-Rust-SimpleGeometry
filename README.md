# 開發背景
DreamPlay(簡稱DP)原本是用Unity與C#編寫的遊戲，主要偏向桌遊平台，像WePlay或Roblox一樣，但因為技術債和我認為這玩法太難開發以及大家或許想要的不是複雜的遊戲，而是簡潔、輕量和與眾不同的玩法，於是後來我正式在2026-02-12暫時中止這項計畫。後來我接觸了Rust、Fyrox、WASM、WASI，給了我重寫遊戲的希望，所以我正式在2026-05-16開始用Rust和Fyrox重寫遊戲。

# DP-Rust-SimpleGeometry是什麼專案?
DP-Rust-SimpleGeometry採用Rust語言開發，並使用Fyrox遊戲引擎開發。

DP-Rust-SimpleGeometry與原本的DreamPlay不同，DreamPlay偏向大型桌遊平台，但這次的DP-Rust-SimpleGeometry選擇走簡約幾何風格，並融入一點桌遊玩法，只不過，未來可能不會只有桌遊玩法。
