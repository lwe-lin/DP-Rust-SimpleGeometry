# 開發背景(Development Background)
DreamPlay(簡稱DP)原本是用Unity與C#編寫的遊戲，主要偏向桌遊平台，像WePlay或Roblox一樣，但因為技術債和我認為這玩法太難開發以及大家或許想要的不是複雜的遊戲，而是簡潔、輕量和與眾不同的玩法，於是後來我正式在2026-02-12暫時中止這項計畫。

後來我接觸了Rust、Fyrox、WASM、WASI，給了我重寫遊戲的希望，所以我正式在2026-05-16開始用Rust和Fyrox重寫遊戲。

DreamPlay (DP for short) was originally a game written in Unity and C#, mainly for board game platforms like WePlay or Roblox. However, due to technical debt, my belief that the gameplay was too difficult to develop, and the fact that people might not want complex games but rather simple, lightweight, and unique gameplay, I officially suspended the project on February 12, 2026.

Later, I came across Rust, Fyrox, WASM, and WASI, which gave me hope of rewriting the game. So, on May 16, 2026, I officially started rewriting the game using Rust and Fyrox.

# DP-Rust-SimpleGeometry是什麼專案?(What is the DP-Rust-SimpleGeometry project?)
DP-Rust-SimpleGeometry採用Rust語言開發，並使用Fyrox遊戲引擎開發。

DP-Rust-SimpleGeometry與原本的DreamPlay不同，DreamPlay偏向大型桌遊平台，但這次的DP-Rust-SimpleGeometry選擇走簡約幾何風格，並融入一點桌遊玩法，只不過，未來可能不會只有桌遊玩法。

DP-Rust-SimpleGeometry is developed using the Rust language and the Fyrox game engine.

DP-Rust-SimpleGeometry differs from the original DreamPlay, which leans towards a large board game platform. DP-Rust-SimpleGeometry, however, adopts a minimalist geometric style and incorporates some board game elements. However, it may not be limited to board game elements in the future.
